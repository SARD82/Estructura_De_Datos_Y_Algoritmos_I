Activities for the **Data Structures and Algorithms I (EDA I)** course at the Faculty of Engineering, UNAM.

## 📘 What You'll Find Here:

- 💻 **In-class activities** in C and Python.
- 🧪 **Lab practice reports** (P1 – P9).
- 📝 **Exams** and mock exams.
- 📁 **Final Project** for the course.
- 📚 **Assignments** and exercise series.

## 🛠️ Technologies & Applied Concepts

- **C language** — Arrays, pointers, dynamic memory (`malloc`, `calloc`, `realloc`, `free`).
- **Python 3** — Support scripts, execution time measurement.
- **Data structures:** Stacks, queues, circular queues, double-ended queues.
- **Sorting algorithms:** Bubble Sort, Quicksort.
- **Recursive algorithms:** Tower of Hanoi.
- **File and string handling** in C.
